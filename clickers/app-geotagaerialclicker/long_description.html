MapKnitter is a demo application for PyBossa that shows how you can crowdsourcing a geo localization problem with MapKnitter as the backend.

This application uses a map from MapKnitter where there are several camp tents. The goal is really simple: navigate the map in order to count the number of tents by adding a point or polygon on top of each tent.

When the user adds a polygon or a point, the action saves the position and the coordinates in GeoJSON format to easily parse it at a later stage.
